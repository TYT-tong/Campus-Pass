## 汇总结论

* 存在通用“类型转换”工具与多种“序列化”实现，覆盖 Web/Jackson/Redis 等关键场景。

## 类型转换相关

* `com.cp.common.core.text.Convert`：强类型安全转换（字符串、数字、布尔、枚举、数组、BigDecimal/BigInteger、半角/全角、金额大写等）。源码：cp-common/cp-common-core/src/main/java/com/cp/common/core/text/Convert.java:1。

* `com.cp.common.core.utils.StringUtils.toInteger`：字符串到 `Integer` 的安全转换。源码：cp-common/cp-common-core/src/main/java/com/cp/common/core/utils/StringUtils.java:639。

* `com.cp.common.core.utils.poi.ExcelUtil`：导入/导出过程中对单元格内容的类型解析与转换（日期、数字、布尔、枚举、集合子对象等），含自定义格式化器 `dataFormatHandlerAdapter`。源码：cp-common/cp-common-core/src/main/java/com/cp/common/core/utils/poi/ExcelUtil.java:1195, 973。

* `com.cp.common.core.utils.JwtUtils`：JWT 的构建与解析（HS512），虽非传统序列化，但属于令牌编码/解码。源码：cp-common/cp-common-core/src/main/java/com/cp/common/core/utils/JwtUtils.java:28,40。

## 序列化相关

* Redis：

  * `FastJson2JsonRedisSerializer`：基于 fastjson2 的 Redis 值序列化/反序列化，启用 `WriteClassName` 与 `JSONReader.autoTypeFilter(Constants.JSON_WHITELIST_STR)` 白名单。源码：cp-common/cp-common-redis/src/main/java/com/cp/common/redis/configure/FastJson2JsonRedisSerializer.java:19,34,44。

  * `RedisConfig`：配置 `RedisTemplate` 的 key/hashKey 使用 `StringRedisSerializer`，value/hashValue 使用上述 fastjson2 序列化器。源码：cp-common/cp-common-redis/src/main/java/com/cp/common/redis/configure/RedisConfig.java:30-38。

* Jackson：

  * `SensitiveJsonSerializer` + 注解 `@Sensitive`：对字符串字段进行数据脱敏的上下文序列化器（管理员不脱敏）。源码：cp-common/cp-common-sensitive/src/main/java/com/cp/common/sensitive/config/SensitiveJsonSerializer.java:21；注解：cp-common/cp-common-sensitive/src/main/java/com/cp/common/sensitive/annotation/Sensitive.java:20。

  * `ApplicationConfig`：`Jackson2ObjectMapperBuilderCustomizer` 统一设置时区。源码：cp-common/cp-common-security/src/main/java/com/cp/common/security/config/ApplicationConfig.java:18。

* 业务对象实现 `Serializable`：

  * `LoginUser`：cp-api/cp-api-system/src/main/java/com/cp/system/api/model/LoginUser.java:21。

  * `R<T>`：cp-common/cp-common-core/src/main/java/com/cp/common/core/domain/R.java:11。

  * `TableDataInfo`：cp-common/cp-common-core/src/main/java/com/cp/common/core/web/page/TableDataInfo.java:11。

  * 以及通用实体 `BaseEntity`（同包路径中）。

## 补充说明

* Redis fastjson2 序列化已启用类型白名单，减少 autoType 风险；若需要进一步安全加固，可限制白名单范围或改用 Jackson 序列化。

* Excel 导入的类型转换与注解驱动（`@Excel/@Excels`）相关，注意日期/数值边界与 CSV 注入防护（以 `= - + @` 开头前置 `\t`）。
